Goal 3: Under-Five Mortality Rate (U5MR) Live Analysis

Interactive spatial mapping of international child health metrics via United Nations SDG monitoring indexes.

Project Documentation & Spatial Insights

Framework: Leaflet.js / JavaScript API Data Source: UN SDG DataHub (2024 Release) Classification: 10-Tier Custom Quantile Breaks

This interactive GIS application evaluates global distribution anomalies regarding target indicators for Sustainable Development Goal 3.2.1. By normalizing global census streams into a unified geographic polygon map layer, the visual output helps pinpoint structural variations across regional healthcare networks.

Methodology & Tooling: The front-end layout relies on standard web map tiles combined with dynamically colored GeoJSON layers. Custom style thresholds allow rapid classification of child mortality indices ranging from standard baselines to critical density clusters concentrated heavily throughout sub-Saharan areas.

User Interactions: Viewers can freely pan and zoom over target vectors. Hovering cursor elements over defined boundaries dynamically recalculates data readouts inside the customized interactive summary panel in the upper corner of the map layout.
